    Solution A.3.

    Note that the hypotheses imply \(ŒΩ(E) = ‚à´_E f \, dŒº\) is a measure, from which the desired conclusion immediately follows.

    Of course, this does not answer the question as stated, since the examiners specifically require the use of the MCT and linearity of the integral.

    Define \(f‚Çô = ‚àë_{k=1}^n f œá_{X_k} = f œá_{‚à™_1^n X_k}\).

    Then it's clear that the hypotheses of the MCT are satisfied. That is, \(‚àÄ x ‚àà X\),

    \(‚àë_{k=1}^‚àû ‚à´_{X‚Çñ} f\, dŒº = \lim_n ‚àë_{k=1}^n ‚à´_X f œá_{X‚Çñ}\, dŒº\\ = \lim_n ‚à´_X ‚àë_{k=1}^n f œá_{X‚Çñ}\, dŒº.\)

    By the MCT, \(‚à´_X \lim_n f‚Çô \, dŒº = ‚à´_X f\, dŒº\).

    Solution A.4.

    By Tonelli's Theorem, if 1. \(f(x,y) ‚â• 0\) is measurable and 2. one of the iterated integrals \(‚à¨ f(x,y)\,dx\,dy\) or \(‚à¨ f(x,y)\,dy\,dx\) exists, then they both exist and are equal.

    Moreover, if one of the iterated integrals is finite, then \(f(x,y) ‚àà L^1(dx,dy)\).

    Let \(g(x, y) = x^{-3/2}\cos(\pi y/2x)\), and apply Tonelli's theorem to the non-negative measurable function \(|g(x, y)|\) as follows:

    \(‚à´_0^1 ‚à´_0^x |g(x, y)| \, dy \, dx = ‚à´_0^1 ‚à´_0^x |x|^{-3/2} \left|\cos\left(\frac{œÄ y}{2x}\right)\right|\,dy\,dx ‚â§ ‚à´_0^1 ‚à´_0^x x^{-3/2}\cdot 1 \,dy\,dx = ‚à´_0^1 x^{-1/2} \,dx = 2\).

    Thus one of the iterated integrals is finite, so \(g(x, y) ‚àà L^1(dx, dy)\) by Tonelli's theorem.

    Therefore, Fubini's Theorem applies to \(g(x, y)\), and yields

    \(‚à´_0^1 ‚à´_y^1 x^{-3/2} \cos \left(\frac{œÄ y}{2x}\right) \, dx \, dy = ‚à´_0^1 ‚à´_0^x x^{-3/2}\cos\left(\frac{\pi y}{2x}\right)\,dy\,dx\).

    The latter is clearly equal to the first (hence the last) of the equivalent expressions below.

    \(‚à´_0^1 x^{-3/2} \cdot \frac{2x}{\pi} \left[\sin\left(\frac{\pi y}{2x}\right)\right]_{y=0}^{y=x} \, dx = ‚à´_0^1 \frac{2}{\pi}x^{-1/2}\, dx = \frac{2}{œÄ} \left[2 x^{1/2}\right]_{x=0}^{x=1} = \frac{4}{œÄ}\).

    Solution A.6

    Obviously both measures are non-negative. We must first prove \(m ‚â™ Œº\). To this end, suppose \(m(E) > 0\), where \(E ‚àà œÉ(m)\), the œÉ-algebra of Lebesgue measurable sets. If we can show \(Œº(E) > 0\), this will establish that the implication \(Œº(E) = 0 \, ‚áí \, m(E) = 0\) holds for all \(E ‚àà œÉ(m)\); i.e., \(m ‚â™ Œº\).

    For \(n = 1, 2, ‚Ä¶\), define \(A_n = \{x ‚àà ‚Ñù : \frac{1}{n+1} < \frac{1}{1+x^2} ‚â§ \frac{1}{n}\}\).

    Then \(A_i ‚à© A_j = ‚àÖ\) for all \(i ‚â† j\) in ‚Ñï, and, for all \(n = 1, 2, ‚Ä¶\), \(Œº(A_n) ‚â• \frac{1}{n+1} m(A_n)\).

    Also, \(‚Ñù = ‚ãÉ_n A_n\), since \(0 < \frac{1}{1+x^2} \leq 1\) holds for all \(x ‚àà ‚Ñù\).

    Therefore, \(Œº(E) = Œº(E ‚à© ‚ãÉ_n A_n) = Œº(‚ãÉ_n (A_n ‚à© E)) = ‚àë_n Œº(A_n ‚à© E)\).

    The last equality might need a bit of justification. Since \(f(x) = \frac{1}{1+x^2}\) is continuous, hence measurable, the sets \(\{A_n\}\) are measurable. Therefore, the last equality holds by countable additivity of disjoint measurable sets.

    Note that \(m(E) = ‚àë_n m(A_n ‚à© E) > 0\) implies the existence of an \(n ‚àà ‚Ñï\) such that \(m(A_n ‚à© E) > 0\). Therefore,

    \(Œº(E) ‚â• Œº(A_n ‚à© E) ‚â• \frac{1}{n+1} m(A_n ‚à© E) > 0\),

    which proves that \(m ‚â™ Œº\).

    By the Radon-Nikodym Theorem there is a unique \(h ‚àà L^1(Œº)\) such that

    \(m(E) = ‚à´ h \, dŒº\), and \(‚à´ f\, dm = ‚à´ f \, h \, dŒº\), for all \(f ‚àà L^1(m)\).

    In particular, if \(E ‚àà œÉ(m)\) and \(f(x) = \frac{1}{1+x^2} œá_E\), then

    \(Œº(E) = ‚à´_E \frac{1}{1+x^2} \, dm(x) = ‚à´_E \frac{h(x)}{1+x^2}\, dŒº(x)\).

    That is, \(‚à´_E \, dŒº = ‚à´_E \frac{h(x)}{1+x^2}\, dŒº(x)\) holds for all measurable sets \(E\), which implies (see note blow) that, \(\frac{h(x)}{1+x^2} = 1\) holds for \(Œº\)-almost every \(x ‚àà ‚Ñù\). Therefore, \(\frac{dm}{dŒº}(x) = h(x) = 1+x^2\).

    ‚àé

    Remarks.

    • The penultimate implication comes from the following standard result: if \(f\) and \(g\) are integrable functions such that \(‚à´_E f = ‚à´_E g\) holds for all measurable sets \(E\), then \(f = g\), \(Œº\)-a.e.

    • \(h\) is uniquely defined only up to an equivalence class of functions that are equal to \(1+x^2\), \(Œº\)-a.e.

    Solution B.2.

    Preliminary remark. I have worked this problem a number of times, and what follows is the clearest and most instructive proof I have come up with. It is by no means the shortest, most elegant solution, and probably not the type of detailed answer one ought to provide on an actual exam, when time is a factor. However, some of the facts that I prove in detail here have appeared as separate questions on other exams, so the proofs are worth knowing.

    Note that \(f\) is increasing, so \(f'\) exists for a.e. \(x ‚àà ‚Ñù\) and \(f'(x) ‚â• 0\) wherever \(f'\) exists. Also, \(f'\) is measurable. To see this, define

    \(g(x) = \limsup_{n ‚Üí ‚àû} \, [f(x+ \frac{1}{n})-f(x)] \, n\).

    As a lim sup of a sequence of measurable functions, \(g\) is measurable (Theorem 1.14 of Rudin (1987)).

    Let \(E\) be the set on which \(f'\) exists. Then \(m(‚Ñù - E) = 0\), and \(f' = g\) on \(E\) (by definition of derivative), so \(f'\) is measurable.

    • (‚áê) Suppose \(‚à´_‚Ñù f' \, dm = 1\). We must show \(f ‚àà AC[a, b]\) for all \(-‚àû < a < b < ‚àû\). First, check that \(f' ‚àà L^1(‚Ñù)\), since

      \(1 = ‚à´_‚Ñù f' \, dm = ‚à´_{‚Ñù - E} f' \, dm + ‚à´_E f' \, dm = ‚à´_E f' \, dm\),

      (The last equality holds because \(m(‚Ñù - E) = 0\).)

      Since \(f\) is increasing, \(f' ‚â• 0\) on \(E\), so

      \(‚à´_‚Ñù |f'| \, dm = ‚à´_{‚Ñù - E} |f'| \, dm + ‚à´_E |f'| \, dm = ‚à´_E |f'| \, dm = ‚à´_E f' dm = 1\).

      Thus, \(f' ‚àà L^1(‚Ñù)\) as claimed.

      A couple of lemmas will be needed to complete the ‚áê direction of the proof. The first is proved in the appendix while the second can be found on page 100 of Royden (1988).

      Lemma 8.1. Let \(f: ‚Ñù ‚Üí ‚Ñù\) be a function. If \(f\) is differentiable on \([a, b]\), \(f' ‚àà L^1([a, b])\), and \(‚à´_a^x f'(t) \, dt = f(x) - f(a)\) for \(a ‚â§ x ‚â§ b\), then \(f ‚àà AC[a,b]\).

      The converse of this lemma is also true.

      See Folland (1999) for a nice treatment of the Fundamental Theorem of Calculus (FTC) for Lebesgue integration.

      Lemma 8.2. If \(f: ‚Ñù ‚Üí ‚Ñù\) is increasing and \(f' ‚àà L^1([a,b])\), then \(‚à´_a^x f'(t) \, dt ‚â§ f(x) - f(a)\).

      To finish the ‚áê direction of the proof, by Lemma 8.1, it suffices to show that \(‚à´_‚Ñù f' \, dm = 1\) implies \(‚à´_a^b f'(t) \, dt = f(b) - f(a)\) holds for all \(-‚àû < a < b < ‚àû\).

      By Lemma 8.2, we have \(‚à´_a^b f'(t) \, dt ‚â§ f(b) - f(a)\), so we need only show that strict inequality cannot hold.

      Suppose, by way of contradiction, that \(‚à´_a^b f'(t) \, dt < f(b) - f(a)\) holds for some \(-‚àû < a < b < ‚àû\). Then,

      \(1 = ‚à´_‚Ñù f' \, dm = ‚à´_{-‚àû}^a f' \, dm + ‚à´_a^b f' \, dm + ‚à´_b^‚àû f' \, dm\)

      \(< [f(a) - f(-‚àû)] + [f(b) - f(a)] + [f(‚àû) - f(b)] = f(‚àû) - f(-‚àû) = 1\).

      Therefore, \(‚à´_‚Ñù f' \, dm = 1\) implies that

      \(‚à´_a^b f'(t) \, dt = f(b) - f(a)\) holds for all \(-‚àû < a < b < ‚àû\), as desired.

    • (‚áí) Assume \(f ‚àà AC[a, b]\) for all \(-‚àû < a < b < ‚àû\). We must show \(‚à´_‚Ñù f' \, dm = 1\). By assumption \(f(‚àû) - f(-‚àû) = 1\), so this is equivalent to showing

      \(\lim_{x ‚Üí ‚àû} ‚à´_{-x}^x f'(t) dm(t) = \lim_{x ‚Üí ‚àû}[f(x) - f(-x)]\).

      Let \(x‚àà ‚Ñù\), \(x > 0\), and \(f ‚àà AC[-x,x]\). We claim \(f(x)-f(-x) = ‚à´_{-x}^x f' dm\). Assuming the claim is true---see p. 110 of Royden (1988) for the proof---we have

      \(1 = \lim_{x ‚Üí ‚àû} [f(x) - f(-x)] = \lim_{x ‚Üí ‚àû}‚à´_{-x}^x f'(t) dm(t) = ‚à´_‚Ñù f' dm\).

    Solution B.3.

    Since \(F ‚àà L^1[-1, 1]^*\), then by the Riesz Representation Theorem there is a unique \(h ‚àà L^‚àû[-1,1]\) such that

    \(F(f) = ‚à´^1_{-1} f(x) h(x) \, dx\) for all \(f ‚àà L^1[-1,1]\).

    Let \(h = œï + œà\), where \(œï\) and \(œà\) are the even and odd functions,

    \(œï(x) = \frac{h(x) + h(-x)}{2}\) and \(œà(x) = \frac{h(x) - h(-x)}{2}\).

    Let \(f = f_e + f_o\) be the decomposition of \(f\) into a sum of even and odd functions. Then, by linearity of \(F\), and since \(F(f_o) = 0\) by hypothesis,

    \(F(f) = F(f_e) + F(f_o) = F(f_e) = ‚à´_{-1}^1 f_e h = ‚à´_{-1}^1 f_e œï +‚à´_{-1}^1 f_e œà\).

    Now note that \(f_e œà\) is an odd function (even times odd), so \(‚à´_{-1}^1 f_e œà = 0\), since \([-1, 1]\) is symmetric. Similarly, \(‚à´_{-1}^1 f_o œï = 0\). Therefore,

    \(F(f) = F(f_e) = ‚à´_{-1}^1 f_e œï = ‚à´_{-1}^1 f_e œï + ‚à´_{-1}^1 f_o œï = ‚à´_{-1}^1 (f_e+f_o) œï = ‚à´_{-1}^1 f œï\).
